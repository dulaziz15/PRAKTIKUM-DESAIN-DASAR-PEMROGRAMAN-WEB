Position: static;

Position digunakan untuk mengatur element menjadi static secara default. Element akan mengikuti posisi normail secara default, element tidak dipengaruhi oleh properti seperti top, bottom, left, dan right.:

Element ini diatur menggunakan posisi: static;




Position: Relative;

Sebuah element HTML yang menggunakan position relative akan terletak pada posisi normal. Mengatur properti diatas, kanan, bawah, dan kiri dari elemen yang diposisikan dengan "relative" akan membuatnya jauh dari poisis normalnya. konten lain tidak akan disesuaikan agar sesuai dengan celah yang ditinggalkan oleh element tersebut,:

element ini diatur menggunakan posisi: relative;



Posiyion : fixed;

Sebuah element HTML yang disetting dengan position fixed akan memiliki sifat tetap, tanpa ada perubahan bahkan jika halaman website discroll. berlaku pengaturan left, bottom, top dan right pada position fixed:

Element ini diatur menggunakan position : fixed















Position: Absolut;

Element HTML yang menggunakan position absolute akan diposisikan relatif dengan elemen lain yang mendahuluinya yang terdekat, bukan relatif terhadap layar secara normal:

Element ini diatur menggunakan posisi: relative;
Element ini diatur menggunakan posisi: absolute;
